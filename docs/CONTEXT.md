## 현재 진행 중: 미니어처 갓 핸드 (인디게임)

### 게임 컨셉

테이블 위 미니어처 디오라마 세상에 사는 작은 주민들의 생활을 관찰하는 힐링 게임. 플레이어는 환경 토글 버튼(비, 햇빛, 바람, 겨울)으로 환경만 바꿔주고, NPC가 자율적으로 적응하는 모습을 관찰.

### 기술 스택 (확정)

- **엔진**: Unity 6 LTS + URP
- **언어**: C#
- **시점**: 고정 앵글 3D (45도, 90도 스냅 회전)
- **배포**: itch.io → Steam

### 현재 Phase: Phase 0 (학습 기초)

| 항목 | 상태 |
|------|------|
| GDD (게임 설계 문서) | ✅ 완료 |
| 엔진 선택 (3+1 합의) | ✅ Unity 확정 |
| C# 기초 문법 학습 | ✅ 완료 |
| C# Unity 패턴 학습 | ✅ 완료 (MonoBehaviour, 싱글톤, 이벤트, FSM) |
| Unity 프로젝트 생성 | ⏳ 대기 (노트북 Unity 연결 필요) |
| Unity 에디터 학습 | ⏳ 대기 |
| NavMesh 학습 | ⏳ 대기 |

- **서버 (Linux)**: Claude Code — 코드 작성, 설계 문서, .NET SDK 설치됨
- **노트북 (macOS)**: Unity Editor — 시각적 작업
