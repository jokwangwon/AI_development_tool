# ADR-005: AI 백엔드 스택 가이드라인

**상태**: 승인 (3+1 합의 완료)
**날짜**: 2026-04-06
**의사결정자**: 3+1 에이전트 합의

---

## 맥락 (Context)

AI 기능이 필요한 프로젝트에서 AI/ML 라이브러리 생태계가 Python에 집중되어 있으므로, AI 백엔드의 기본 언어 가이드라인이 필요하다.

## 결정 (Decision)

**로컬 추론/학습이 필요하면 Python AI 백엔드를 분리한다. API 호출만이면 분리 불필요.**

- 분리 트리거: "AI 기능" 아닌 **"로컬 추론/학습 필요"** (2단계 분기)
- 기본 스택: Python >= 3.11, FastAPI, uv, ruff, pytest
- 보안 최소 세트: 키 관리 + 출력 검증 + 인젝션 방어
- 기본값(가이드라인)이지 절대 규칙 아님. 예외 시 ADR 필수

## 3+1 합의 결과

| 에이전트 | 핵심 | 반영 |
|---------|------|------|
| Agent A | 분리 기준 모호, 운영 부재 | 2단계 분기 + Docker 참조 |
| Agent B | 보안 전무, 예외 형해화 | 보안 3항목 + 예외 기준 강화 |
| Agent C | "규정"→"가이드라인", API/로컬 분기 | 톤 조정 + 2단계 분기 |
| **Reviewer** | **C 프레임 + B 보안 + A 실용성** | 통합 |
